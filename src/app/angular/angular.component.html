1. Architecture

1.1 Overview

  • NgModules

    The main building blocks of an Angular project are NgModules. NgModules collects related code in functional blocks, it is a container for cohesive block of codes, a workflow, ... . A Module can contain Components, service providers and other code files.

    A Module can export his own functionality for others to use and can import other modules for their functionality.

    An App always has at least one root module the enables bootstrap.

  • Components

    Components define views. Set of screen elements that can be renderend on behalf of your programs logic and data. A component can render and change the DOM (Document Object Model).

  • Services

    Services provide functionality that a view does not need to know. Services are injected into Components as dependencies.
